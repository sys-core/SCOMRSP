Bienvenue à la page SCOMRSP.

Nous vous présentons une application qui recueille et enregistre le traffic TCP ou UDP isolant uniquement les combinaisons Source IP, le port source, l'IP de destination, et le port de destination. Cette application en Python peut être contrôlé par des arguments de ligne de commande afin de l'intégrer facilement avec les tâches "cron" ou d'autres mécanismes de planification.

Comment l'utiliser

SCOMRSP est conçu pour être utilisé comme une commande comme suit:

sudo Python SCOMRSP –v –TCP –m 60 –p /home/chet/Desktop

La commande accèpte les arguments suivants:

-v Verbal: Écrit les résultats intermédiaires sur la sortie standard lorsque cette option est spécifiée -m Minutes: Précise la durée en minutes pour effectuer les activités de collecte -TCP|-udp Protocole : Définit le protocole d'application à capturer -p Produit: Définit le répertoire de sortie pour le journal légale et le fichier csv
